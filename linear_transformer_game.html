线性注意力原理

线性注意力核心思想
移除 softmax 操作符，利用矩阵乘法的结合律实现计算重排：
𝐨t = ∑j=1t (𝐪t⊤𝐤j)𝐯j = (∑j=1t 𝐯j𝐤j⊤) 𝐪t
递归状态定义
定义状态矩阵：
𝐒t = ∑j=1t 𝐯j𝐤j⊤ ∈ ℝd×d
递归更新公式：
𝐒t = 𝐒t-1 + 𝐯t𝐤t⊤
𝐨t = 𝐒t𝐪t

线性注意力的核心创新在于将原本需要维护整个历史序列（复杂度 O(L)）的注意力机制，转换为维护一个固定大小的状态矩阵 𝐒t（复杂度 O(d²)）。

在递归过程中：

  • 每个时间步 t 接收新的 (𝐪t, 𝐤t, 𝐯t) 三元组
  • 计算外积 𝐯t𝐤t⊤（维度 d×d）
  • 将外积累加到状态矩阵 𝐒t = 𝐒t-1 + 𝐯t𝐤t⊤
  • 输出 𝐨t = 𝐒t𝐪t

这种递归形式使线性注意力本质上成为一个线性RNN，具有固定大小的状态，显著降低了自回归生成时的计算复杂度。

复杂度分析
标准注意力: O(L²d) 训练, O(Ld) 推理
线性注意力: O(Ld²) 训练, O(d²) 推理
